comandos pra instalar o breeze

1-trazendo as dependecias do breeze pra dentro do projeto  

   composer require laravel/breeze

2 - aplicando o comando de instalação

   php artisan breeze:install

3 - escolher a bliblioteca visual de controle 
   
   eu escohi o blade

4 -Escolher se ele vai ter a funcionalidade dar mode 

   eu disse que não 

5 - escolher libs de penteste

   eu coloquei 0 porque não quero fazer issso agora 

6 - seed o laravel cria um seed com uma usuario padrao para salvar 
    na base de dados e eo comando abaixo executa esse seed,,salvando o usuario na base de dados    
  
   php artisan db:seed --class=DatabaseSeeder
