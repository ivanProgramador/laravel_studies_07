para fazer isso tem que acessar o model user qque tem uma linha comentada 
  
  use Illuminate\Contracts\Auth\MustVerifyEmail; 

ao remover as barras de comentario essa importação ficara ativa 
depois configurar ela pra ser implementada na classe 

assinatura classe normal
-----------------------------------------------------------
 class User extends Authenticatable{}


depois da implementação 
--------------------------------------------------------
class User extends Authenticatable implements MustVerifyEmail {}


depis de fazer esse dois passos um cliente de emial deve ser definido no .env 
nessse caso se sair um emial do sistema isso ficara registrado no log 

--------------------------------------------------
MAIL_MAILER=log

---------------------------------------------------------------------
Depois de fazer isso todo o cliente que logar recebera um emial de confirmação sendo 
que o sistema so vai deixar ele passsar para a dashboard depoiss de confirmar o emial.
